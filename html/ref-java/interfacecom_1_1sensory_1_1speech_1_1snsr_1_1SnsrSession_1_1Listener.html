Detailed Description

Event listener interface for the SnsrSession class.

This interface contains a single method, onEvent() that is invoked by the run() and forEach() methods.

SnsrRC onEvent ( SnsrSession  s,
String  key 
)

SnsrStream event handler method.

This method is invoked by a SnsrSession instance to post events to the caller. The key parameter matches the key used to register the event (setHandler()) or start the iterator (forEach()).

Event callbacks happen in the same thread that the forEach() or run() method was called in. Processing is paused until this method returns, best practice is therefore to have this method return as quickly as possible.

Return SnsrRC.OK to continue processing, or any other SnsrRC value to stop.

Return codes other than SnsrRC.OK, SnsrRC.STREAM_END, SnsrRC.STOP, SnsrRC.SKIP, SnsrRC.REPEAT, and SnsrRC.TIMED_OUT will cause run() or forEach() to throw an exception.

See Task Descriptions for a list of task-specific event and iterator keys

sthe SnsrSession instance that is invoking the method.
