Conceptual Model

Overview of SDK concepts.

This library uses a dataflow approach to evaluate speech recognition tasks. It relies on inversion of control: User-defined hooks are invoked to report results and control task flow. Task models define flow pipelines and behavior. The API contains two primary data types: SnsrSession and SnsrStream. Sessions encapsulate the entire state of a task instance, and all session input and output data are are vectored through streams.

Session instance behavior is customized via settings, of which there are three variants:

  • Configuration settings are serializable read-write values. These are used for parameter values and model objects. Task models contain reasonable default values - modify these only to tweak results.
  • Runtime settings are instance-specific and not serializable. These include callback handles, iterators, input streams and values such as the name of the phrase being enrolled.
  • Result settings are read-only values with very limited scope, such as the recognition results.

Stream instances are created on various underlying data streams, such as FILE * handles, memory segments, and live audio sources. Only the constructors are type-specific, all other functions are independent of the underlying stream type. Operations that take stream handles as arguments operate on any of the types.

Task evaluation typically follows this recipe:

  • Create a new session instance.
  • Load a task model into the instance.
  • Set the input source stream.
  • Register one or more event handlers.
  • Enter the main loop. The library will process the input streams and invoke event handlers at appropriate times. The main loop continues until a terminating condition is reached, such as an event returning an error code.
  • Release the session instance.

Java phrase spotter

SnsrSession s = new SnsrSession();
try {
s.load("spot-hbg-enUS-1.4.0-m.snsr") // This is a phrase spotter model
.require(Snsr.TASK_TYPE, Snsr.PHRASESPOT); // Verify the model type
} catch (IOException e) {
e.printStackTrace();
System.exit(3); // A hard exit is not recommended for production use.
}
s.setStream(Snsr.SOURCE_AUDIO_PCM, SnsrStream.fromAudioDevice()) // live audio
// Print the recognition result from a result event
.setHandler(Snsr.RESULT_EVENT, new SnsrSession.Listener() {
public SnsrRC onEvent(SnsrSession s, String key) {
System.out.println(String.format("\"%s\", score = %.3f",
s.getString(Snsr.RES_TEXT),
s.getDouble(Snsr.RES_SV_SCORE)));
return SnsrRC.STOP; // Stop after a single spot.
}
});
try {
s.run(); // Enter the main loop.
} catch (IOException e) {
e.printStackTrace();
}
s.release(); // Optional, releases the native library handles immediately.
s = null;
