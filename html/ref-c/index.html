Overview

This is release 7.4.0 of Sensory's TrulyNatural speech recognition SDK, which includes:

  • A simplified C API with support for live audio capture on iOS, macOS, Linux, and Windows.
  • Running Large Vocabulary Continuous Speech Recognition tasks.
    • Includes tools to build LVCSR models from grammars or lists of phrases,
    • and a model suitable for on-device command and control.
    • VoiceHub provides a user interface for creating recognition grammars.
  • Fixed-phrase and enrolled phrase spotter support. Create custom spotter models with VoiceHub.
  • Command line tools to enroll and evaluate phrase spotter models.
  • Model output suitable for use with Sensory's DSP phrase spotter products.
  • See Changelog for a detailed list of changes from previous releases.

The TrulyNatural SDK is a strict superset of Sensory's TrulyHandsfree SDK, with additional support for large vocabulary speech-to-text.

Requirements

  • macOS, x86_64 Linux, or Windows (version 10 or later, Microsoft Visual Studio 2022) development machine.
  • iOS: Xcode 15.0 or later.
  • For optimal performance, ensure that the dynamic range of the input audio spans at least 12 bits (-24 dBFS peak-to-peak, sample values from -2048 to 2047) and that no clipping is present.

Performance

  • TrulyNatural libraries for Sensory's standard ports are fully optimized. Vector instructions are used where available on x86 and ARM CPUs.
  • Much lower MCPS can be achieved by using Neon instructions when available on ARM cores.
  • Sensory Sales can assist in selecting appropriately sized models for your target platform.
