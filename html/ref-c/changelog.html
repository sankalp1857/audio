  • Added

      • Includes Open Source modules, see Open Source Licenses.

    • snsr-edit -p option reduces SNSR_PHRASESPOT model sizes by removing operating points other than the currently selected SNSR_OPERATING_POINT.
  • Changed
    • The deprecated SNSR_SPOT_DELAY setting is read-only.
    • DSP library (for creating embedded spotter models) version 7.4.0.
    • ‡ The effects of : terminal rewrites are now visible in SNSR_RESULT_EVENT SNSR_RES_TEXT values.
    • ‡ The lightweight NLU processor scoring algorithm no longer considers :inserted and remove: words when ranking result parses. This potentially changes the primary NLU result, or the order in which NLU results are presented when SNSR_NLU_RES_MAX > 1, for grammars that use the : rewrite feature to better match expectations.
    • ‡ If the time required to compute a partial LVCSR result exceeds the SNSR_PUSH_DURATION_LIMIT, the SNSR_PARTIAL_RESULT_EVENT callback does not happen.
    • ‡ The Custom Recognition grammars support the literal use of special tokens by escaping them with a backslash.
  • Deprecated
    • Lightweight VAD. Use a Voice Activity Detector instead.
  • Removed
    • ‡ References to the <- slot assignment operator in Custom Recognition. This had been a symbol reserved for internal use only.
    • ‡ lvcsr-broad-enUS-3.1.0-t5.snsr
  • Fixed
    • Machine-learned Voice Activity Detector models could unexpectedly report an error that
      from-index (...) must be >= first-index (...)
    • A subset of spotter models showed increased false accept rates when used with Task Templates
    • ‡ tpl-opt-spot-vad-lvcsr-model mistakenly invoked SNSR_SILENCE_EVENT and SNSR_LISTEN_END_EVENT multiple times at end-of-stream if SNSR_SLOT_0 SNSR_INCLUDE_LEADING_SILENCE was set to @ 1.
    • ‡ tpl-spot-concurrent with a spotter model in SNSR_SLOT_0 and an LVCSR model (gated by a VAD, or spotter and VAD) in SNSR_SLOT_1 would mistakenly return SNSR_RES_NLU_SLOT_NAME and SNSR_RES_NLU_SLOT_VALUE values as NULL.
    • ‡ SNSR_NLU_SLOT_EVENT could still occur even if SNSR_COMPLETE_ONLY suppressed the recognition result.
    • ‡ Class-based recognition could fail with a segmentation fault when attempting to build a recognizer with too many class references.
  • Security
    • This sequence of operations could lead to snsrGetString() returning an unterminated string:
      • Use snsrSetStream() to set a configuration value to a sequence of bytes that does not include \0.
      • Save the model to persistent storage.
      • Load the modified model from persistent storage.
      • Access the configuration value with snsrGetString().
